Tkinter2 file for Keyword Extraction.

The initial keyword frequency check is done on Excel using the command: INDEX(A1:A10, MODE(MATCH(A1:A10, A1:A10, 0)))

The interface file is the main Python UI File. The user can directly load a .txt file and run tokenisation, Named Entity Recognition, Sentence Extraction, and Dependency Scoring step by step.

Thesisthreat.txt is a sample file with data extracted from threat analysis results.
